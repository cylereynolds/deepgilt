# Character portraits

Drop your class portrait renders here as `<class-id>.png` and the character-select
roster (`classes.html`) shows them automatically — it falls back to the pixel sprite
if a file is missing, so you can add them one at a time.

Expected files:

    bonewright.png   reaver.png   pyre.png   warden.png   stalker.png   feral.png

Any resolution works (drawn cover-fit into the card). Front-facing full-body on a
dark or transparent background looks best — the same framing the Leonardo prompt
produces. These are concept/portrait art; in-game sprites are a separate (Blender /
render-to-sprite) pipeline.
